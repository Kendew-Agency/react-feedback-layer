# Product Overview

**@kendew/react-feedback-layer** is a drop-in React component library for collecting contextual user feedback through interactive comments and annotations.

## Core Features

- Visual feedback collection with positioned comments on any UI element
- Comment lifecycle management: draft → published → resolving → resolved
- User attribution with timestamps
- Fully customizable component composition using compound component pattern
- TypeScript-first with complete type definitions
- Minimal dependencies (only React 18+ and Radix UI Slot)

## Key Use Cases

- Design review and collaboration tools
- User feedback collection on prototypes
- Bug reporting with visual context
- Annotation systems for content review

## Architecture Philosophy

- Headless UI approach - provides logic and structure, not styling
- Compound component pattern for maximum flexibility
- Context-based state management for comments
- Reducer pattern for predictable state updates
